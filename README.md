# StudentCare-Assistant#
### “Your smart support for school life.”

StudentCare Assistant — бесплатный многоязычный AI-помощник для школьников 5–11 классов, доступный на трёх языках: русский, казахский и английский.

---

## Почему этот проект важен?

- Помогает с учебой и домашними заданиями  
- Поддерживает в социальных и эмоциональных вопросах  
- Объясняет сложные темы простым языком  
- Улучшает мотивацию и самодисциплину  
- Доступен для школьников Казахстана и СНГ

---

## Основные функции

- Многоязычный чат-ассистент  
- Помощь со стрессом и социальной адаптацией  
- Учебные советы и поддержка  
- Страница с отзывами и статистикой использования

---

## Технологии

- HTML, CSS, JavaScript  
- GitHub Pages для хостинга сайта  
- JSON для хранения данных (отзывы, статистика)  
- Возможное подключение AI API в будущем

---

## Как использовать?

Перейдите по ссылке на [StudentCare Assistant](https://Rasul-sys.github.io/StudentCare-Assistant/) и начните общение с ИИ-ассистентом.
